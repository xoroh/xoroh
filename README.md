# Xoroh

Welcome to the official open-source monorepo for the **Xoroh Platform**. We are dedicated to building a modern, scalable, and open-source infrastructure for the next generation of digital services.

> [!IMPORTANT]
> This project is in its early stages. We are currently initializing the workspace and setting up the core modules.

## 🚀 Getting Started

This repository is powered by [Nx](https://nx.dev).

### Prerequisites

- [Bun](https://bun.sh) runtime
- Node.js (v20+)

### Installation

```bash
# Clone the repository
git clone https://github.com/xoroh/xoroh.git

# Install dependencies
bun install
```

## 🏗️ Structure

- `packages/`: Core libraries and shared utilities.
- `apps/`: Platform components and services (coming soon).

## 🛠️ Development

To explore the workspace graph:

```bash
bun nx graph
```
